# Ethicli
Browser extension for ethical online shopping

# Dev Setup
requires node
requires maven
backend uses java11
`npm install tldts-experimental`
`npm install snowpack`

# Build
npx snowpack build

# Run
Install extension from ExtensionBuild directory
